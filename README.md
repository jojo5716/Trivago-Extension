# Trivago-Extension
----
Steps to clone repository:

1.- git clone https://github.com/jojo5716/Trivago-Extension

2.- cd Trivago-extension

3.- npm i

4.- go to chrome extension and add this folder


5.- go to Trivago.es and make a search.

6.- Enjoy! =)
